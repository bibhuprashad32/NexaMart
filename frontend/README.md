# NexaMart Frontend

This is the user interface for the NexaMart E-commerce application, built using React and Bootstrap.

## Tech Stack

* **React.js** (Bootstrapped with Create React App)
* **Bootstrap:** For responsive layout and UI components.
* **React Router DOM:** For client-side routing between pages.
* **Axios:** For making HTTP requests to the Spring Boot backend.
* **React Toastify:** For displaying elegant success and error notifications.

## Features

* **Customer Portal:** Browse products by category, manage cart, place orders, and view order history.
* **Admin Dashboard:** Interface to add categories, add products, view all orders, and assign delivery personnel.
* **Delivery Dashboard:** A dedicated view for delivery staff to check assigned orders and update delivery statuses.
* **Authentication UI:** Login, Registration, and Forgot Password flows (with OTP verification).

## How to Run Locally

### Prerequisites
* Node.js and npm installed on your machine.
* The Spring Boot backend server must be running for the UI to display data correctly.

### Steps
1. Open your terminal and navigate to the `frontend` folder.
2. Install the dependencies: `npm install`
3. Start the development server: `npm start`

The app will automatically open in your browser at http://localhost:3000.
